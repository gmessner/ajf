Resource Loading Using "A" Java Framework (AJF)

ResourceBundle Description

ResourceBundles are used to store resources for an application that do not change as the application runs. These resources are usually things like Strings that are displayed on labels and buttons throughout the application. They may also specify other resources such as URLs for icons.

Each ResourceBundle is associated with a particular class and the default system Locale and is automatically loaded when an instance of com.messners.ajf.ui.ResourceLoader is created. Usually only top-level frames and dialogs will have a ResourceBundle associated with them, with framework components being an exception. To create an instance of ResourceLoader for your top-level frame or dialog simply create a ResourceLoader instance with the the Class of the top-level frame or dialog as the only parameter:

    ResourceLoader resLoader = new ResourceLoader(this.getClass());

Loading JComponents

JComponent resources are loaded by a ResourceLoader instance using the JComponent load(String name) method, where name specifies the name of the resource in the ResourceBundle. For example, a button resource could be defined in the ResourceBundle that specifies a JButton component with "Next" as the label and a right pointing arrow as the icon, the resource definition would look similar to this:

    next.type=button
    next.label=Next
    next.icon=RightArrow.gif

You would then load this resource as follows:

    JComponent nextButton = resLoader.load("next");
	
    or
	
    JButton nextButton = (JButton)resLOader.load("next");

Resource attributes are described as follows:

Attribute Description
actionFully qualified class name for the Action. Actions are only applicable if the resource is on a menu or toolbar.
alignmentSpecifies label alignment. Must be right, center, left, leading, or trailing.
borderSpecifies the border type. Must be none, etched, raised-bevel, lowered-bevel, line, or titled. If border="titled" or border="none", use the "title" or "label" resource for the title string to be displayed.
editableSpecifies whether the text field is editable. Default is true for text-field and false for combobox.
iconThe icon URL for the resource.
itemsSpace delimeited list of items to be placed in the component. For a combobox this would be the list of strings for the popup menu, for a menubar this is a list of top-level menus, for a menu or popup menu this is a list of components to be placed on the menu, and for a toolbar it is a list of toolbar components.
labelThe label text for the label. Labels may contain a mnemonic which is identified by a character being preceeded by an & (ampersand).
maskThe input mask for the text field, see JFormattedTextField and MaskFormatter for more info.
orientationSpecifies the orientation of the toolbar.
rolloverSpecifies whether the radio button should be displayed using rollover effects when displayed on a toolbar.
shortcutShortcut to activate the component. Shortcuts are specified using either Alt+key or Ctrl+key notaion.
tooltipThe tooltip text for the component. Can contain HTML.
typeSee the matrix above.
valid-charsRestricts valid characters, only used in conjumction with mask.
valueThe value to assign to this radio button, this value will be returned by calling getSelectedValue() on the containing com.messners.ajf.ui.ButtonGroupExt. Defaults to the name of the radio button.
widthThe width in columns to use when computing the preferred size of this component.


Loading JTextFields

JTextFields resources are loaded by a ResourceLoader instance using the text-field type. This type has the following special attributes:

Attribute Description
alignmentSpecifies text alignment. Must be right, center, left, leading, or trailing.
documentSpecified the Document type, must either be null or one of the following:
alpha-numeric, double, integer.
editableSpecifies whether the text field is editable. Default is true for text-field and false for combobox.
maskThe input mask for the text field, see JFormattedTextField and MaskFormatter for more info.
max-valueSpecifies the maximum value for a double or integer document type.
max-widthSpecifies the maximum number of characters for the field.
min-valueSpecifies the minimum value for a double or integer document type.
valid-charsRestricts valid characters, only used in conjumction with mask.
valueThe value to assign to the text-field.
widthThe width in columns for the text-field.


Loading JPanels

JPanel resources are loaded by a ResourceLoader instance using either a group or radio-group type. This type has the following special attributes:

Attribute Description
columnsThe number of columns for the layout of the JPanel.
headingSpecifies a label to put to the left of the managed components.
itemsA list of component names to add to this JPanel.
layoutSpecifies the layout manager for this JPanel. Currently only "labeled-pair" is supported.
rowsThe number of rows for the layout of the JPanel. Will be automatically computed if not specified.
